# Rihlah Travel
A simple Rihlah Travel web application that allows users to book Umrah and Hajj packages online. Users can sign up, select their preferred package, fill in travel and personal details, upload a payment screenshot, and finalize their booking.
Built using HTML, CSS, and JavaScript
Stores data locally using localStorage
Fully responsive and works on desktop and mobile
Dark-and-gold modern theme, easy to use
## Live Demo
https://semirakemal.github.io/Rihlah-Travel/

## Features
User registration and login system
Booking details form for travel date, number of pilgrims, room type, and passport number
Package selection: Umrah Essential, Umrah Premium, or Hajj Package
Upload payment screenshot to confirm payment
Booking summary page displaying all details
Finish Booking button with confirmation alert
Fully responsive design for mobile and desktop
## Technologies Used
HTML for structure
CSS for styling
JavaScript for interactivity, form validation, and localStorage
Font Awesome for icons
